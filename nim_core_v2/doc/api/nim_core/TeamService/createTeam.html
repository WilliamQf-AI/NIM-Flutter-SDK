创建一个群组 云信群组分为两类：普通群和高级群，两种群组的消息功能都是相同的，区别在于管理功能。普通群所有人都可以拉人入群，除群主外，其他人都不能踢人； 高级群则拥有完善的成员权限体系及管理功能。创建群的接口相同，传入不同的类型参数即可。

fields ,NIMTeamFieldEnum 群组预设资料, key为数据字段，value对应的值，该值类型必须和field中定义的fieldType一致。 type ,NIMTeamTypeEnum 要创建的群组类型 members 邀请加入的成员帐号列表 可以设置回调函数，如果成功，参数为创建的群组资料
